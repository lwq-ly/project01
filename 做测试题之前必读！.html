    每种技术入门和进阶各读一本经典书，先快速通读，对书上练习都要在电脑上敲一遍代码，然后根据知识点进行大量练习，不懂看书或者Google

HTML基础：¶

  • 菜鸟的 HTML 教程 ，熟悉各种常用标签的用法，尤其是标题、段落、链接、图像、表格、列表、表单、区块、布局、CSS、脚本等
  • 不要求对每个标签的每个属性都了如指掌，知道什么情况下用什么标签就足矣，死记硬背也没什么意义，后面用的多了自然就熟了
    练习：按 http://ife.giscafer.com/ 的小薇学院部分练习做完，完成百度练习题：HTML、CSS基础
  • 推荐书籍：《HTML5与CSS3基础教程》

CSS基础：¶

  • 菜鸟的 CSS 教程 ，熟悉 CSS 语法和选择器、样式、框模型、定位等模块的内容
  • 重点关注选择器语法、选择器优先级计算，熟悉一下常见的2列3列自适应布局方案，各种居中，各种行列的实现方式有哪些，响应式布局的实现方式等。学习CSS布局   总结   实践
  • 相比较 HTML 来说，CSS 重要很多，大致划分为：选择器、数据类型（单位）、文本 & 字体、布局盒模型、动画，包括两方面 transition（过渡） 和 animation（时间轴和关键帧）

JavaScript基础：¶

  • 廖雪峰的 JS 教程 ，包括数据类型、变量、运算符、控制流、函数、对象等；另外，也要熟悉 DOM；BOM 简单了解下就可以了，使用场景不多
  • 按 http://ife.baidu.com/course/all 练习做完，掌握JavaScript基础知识，能够使用JavaScript编写一些复杂度不大的交互功能。
  • 推荐书籍：《JavaScript DOM 编程艺术 》 进阶：《JavaScript权威指南》

HTML5：¶

  • 菜鸟的 HTML5 教程 HTML5 的新特性肯定要了解，内容其实也不是很多，核心的就是 canvas、svg、对多媒体的支持、Web 存储、应用缓存等

CSS3：¶

  • 菜鸟的 CSS3 教程 也是要熟悉那些新特性，最核心的就是弹性盒子
  • 推荐书籍：《CSS揭秘 》 进阶学习使用
